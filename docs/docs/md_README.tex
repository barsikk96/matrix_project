Этот проект представляет собой библиотеку для работы с матрицами на языке C. Включает основные операции, такие как сложение, вычитание, умножение, транспонирование, вычисление детерминанта и работу с файлами.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Структура проекта}{Структура проекта}}\label{md_README_autotoc_md2}
